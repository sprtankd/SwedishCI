# 🇸🇪 Svenska CI — Comprehensible Input for SFI C & D

Master Swedish through **reading and listening** — the natural way.

Graded stories, news articles, listening exercises, and exam simulators — all designed
around Stephen Krashen's **Comprehensible Input** hypothesis.

## Quick start
Visit the hosted version or open **`index.html`** in any browser. No server needed.

## Modes

| Mode | What it does |
|------|-------------|
| 📖 Graded Reader | Short stories at A2→B1+ with glossed vocabulary, audio, and comprehension questions |
| 📰 News Reader | Simplified news articles for SFI D reading practice |
| 🎧 Listening Lab | Audio passages with questions, transcript reveal, and speed controls |
| 🔗 Collocations | Particle verbs and prepositions in context — always in sentences |
| 🧩 Connectors | Discourse connectors (dessutom, däremot, alltså…) in gap-fill texts |
| 📊 Vocab Tracker | Visual dashboard of all words by encounter status and CEFR readiness |
| 📝 Reading Exam | Timed SFI C/D reading comprehension practice tests |
| 🎙️ Listening Exam | Timed SFI C/D listening comprehension practice tests |

## How it works
- **Glossed text:** Tap/hover any unfamiliar word for an instant English translation + audio.
- **Encounter tracking:** Words graduate from "new" → "met" → "familiar" → "known" based on repeated passive encounters.
- **Level progression:** Content is tagged A2-low → A2-high → B1 → B1+.
- **Exam simulation:** Practice under timed SFI C/D conditions to build test confidence.
- **Offline support:** After the first visit, the app works fully offline (airplane mode).
